# Restaurant Page

A multi-page restaurant website built dynamically with JavaScript — no written HTML.
![Uploading Restaurant Home.png…]()

## Overview

A demo website for "Strangers Grill", a fictional restaurant based in Mombasa, Kenya. 
The entire DOM is manipulated and rendered using JavaScript, with page content 
switching dynamically without any page reloads.
![Uploading Restaurant Menu.png…]()

## Features

- Dynamic page rendering — Home, Menu, and About pages built entirely with JS
- Menu page with food items, images, and prices (Pilau, Chapati, Chai, Kuku and more)
- About page with contact information
- Full-page background image with overlay text
- ES6 modules for clean code organisation
- Bundled with Webpack

## Built With

- JavaScript (ES6 Modules)
- HTML
- CSS
- Webpack
